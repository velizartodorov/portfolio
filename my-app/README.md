# Velizar's portfiolo 👨‍💼

React-based single page web application representing Velizar's portfolio.

## How to start? 🤔

In project directory, run in terminal:

```js
npm start
```

Go to <http://localhost:3000/>.

That's it. Have fun! 😎 🎉
